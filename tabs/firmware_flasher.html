Attempt to flash the board automatically (triggered by newly detected serial port).
 
Flashing APJ / .apj files
  1. Remember, only '.apj' files can be flashed by the following steps:
  2. If connected to mavlink,etc. Disconnect.[blue circle top-right]
  3. choose you board type in first dropdown.[above]
  4. Choose one of the options ending in .apj from the second drop-down. [above]
  5. Ensure 'Flash on Connect' is enabled. [above]
  6. Press [Load Firmware [Online] from bottom of page. Release info should display. [below]
  7. Unplug USB from your board, and plug it back in.[This uses your a bootloader, no need to hold button]
  8. Watch the grey progress bar. it should progress thru 'Erase' -> 'Write' -> 'Verify' ->'Success'[below]
  9. You might needs to Unplug USB from your board, and plug it back in, a second time.[sorry]
  10. Connect to mavlink,etc. Connect.[blue circle top-right]
  11. If APJ flashing doesn't work, it might be that you don't have a bootloader, so try DFU instructions with a _with_bl.hex file , just once.
   Flashing DFU / _with_bl.hex files.
  1. Remember, only '_with_bl.hex' files can be flashed by the following steps:
  2. If connected to mavlink,etc. Disconnect.[blue circle top-right]
  3. choose you board type in first dropdown.[above]
  4. Choose one of the options ending in _with_bl.hex from the second drop-down. [above]
  5. Ensure 'Flash on Connect' is enabled. [above]
  6. Press Load Firmware [Online] from bottom of page. Release info should display. [below]
  7. Unplug USB from your board, hold down DfU/boot button, and and plug it back in. [This uses DFU mode, so u MUST hold down tiny button on board while plugging in]
  8. At this point the Serial-select drop-down MUST be displaying 'DFU' to continue.[top right near 'Connect']
  9. Press '[DFU Only] Flash Firmware' ( only for _with_bl.hex file/s)
  10. Watch the grey progress bar. it should progress thru 'Erasing' -> 'Flashing' -> 'Verifying' ->'Programming SUCCESSFUL'[below]
  11. You might needs to Unplug USB from your board, and plug it back in, a second time.[sorry]
  12. Connect to mavlink,etc. Connect.[blue circle top-right]
  13. If this DFU process does not work for you, you might need to add udev rules and the user to plugdev group as described here: https://github.com/martinbudden/betaflight/blob/master/docs/USB%20Flashing.md
